# ADR-0006: Тема оформления через CSS-токены

**Статус:** принято
**Дата:** 2026-07-13

## Контекст

Дизайн поставляется в светлой и тёмной темах. Приложение должно поддерживать три
режима: `system` (по системным настройкам), `light`, `dark` — с переключением из
экрана настроек. Стили уже используют CSS-переменные ([ADR-0001](0001-css-approach.md)),
что делает CSS-токены естественной точкой расширения.

Силы:
- Нужен явный выбор темы, переживающий перезагрузку (хранится в настройках).
- Режим `system` должен реагировать на смену системной темы «на лету».
- `Board`/`Cell` уже завязаны на CSS-переменные — не хочется трогать их модульные стили.

## Решение

Токены темы объявляем в `src/theme/tokens.css`:
- `:root` — светлая тема (значения по умолчанию).
- `[data-theme="dark"]` — явная тёмная.
- `@media (prefers-color-scheme: dark) :root:not([data-theme])` — системная тёмная.

Приём: для режима `system` атрибут `data-theme` **снимается** с `<html>` (иначе
media-query перекрывается явным селектором и перестаёт работать). Для `light`/`dark`
атрибут проставляется. Логика — в `src/theme/useApplyTheme.ts` (`applyTheme(theme)`).

`src/index.css` служит мостом: существующие CSS-переменные `Board`/`Cell`
переопределяются на новые токены, чтобы не менять `Board.module.css`.

Палитра (из `design/Sudoku Screens Standalone.dc.html`):
- **Светлая:** bg `#eef1f7`, card `#fff`, accent `#2f6fed`, ink `#1f2633`;
  клетки — sel `#a9caf6`, same `#cadff9`, peer `#e9f0fc`.
- **Тёмная:** bg `#0e1320`, card `#171d2b`, accent `#4d86f7`, ink `#eef1f8`;
  клетки — sel `#2c4a78`, same `#223a5f`, peer `#1a2233`.

## Последствия

- Тема переключается сменой одного атрибута на `<html>` — без перерисовки дерева React.
- Режим `system` работает через нативный media-query; ручной выбор его перекрывает.
- `Board.module.css` остаётся нетронутым — вся связка идёт через `index.css`.
- Старый блок `@media (prefers-color-scheme: dark)` в `index.css` удаляется —
  его роль переходит в `tokens.css`.
- Закрывает backlog-фичу «Темы оформления» в части светлой/тёмной/системной.
